# Sales and billing system

## Instalation

- Install dependencies composer
```
composer install
```
- Install dependencies npm
```
npm install && npm run dev
```
- Configure enviroment variable
 `in the file .env `
- Migrate database
```
php artisan migrate
```
- Generate key aplication
```
php artisan key:generate
```
- Raise Servidor
```
php artisan serve
```
